This Datagrid can be used as a replacement to the normal HTML table and can be used in Angular2+ web applications. It is light weight, puts less load on the DOM and comes with many features. Following are the list of available features

  1. Uses VirtualScroll be default
  2. Equipped with Toolbar which has Inbuilt search for grid data
  3. Download CSV and Excel files of grid data
  4. Columns can be shown/hidden dynamically
  5. Equipped with Pagination
  6. De-coupled grid styling with a few options
  7. Single/Multi-Select rows in grid
  8. Supports shift-key selection of rows
  9. Can pre-select rows on load of grid
  10. Columns can be shown/hidden dynamically
  11. Text for individual columns can be aligned either left or right
  12. Each/all columns can be made editable
  13. Cell Templates can be assigned for each column
  14. Sorts the column data using merge sort algorithm
  15. Each column can be defined with individual width
